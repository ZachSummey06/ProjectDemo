Rowdy Rhythms
Rowdy Rhythms is a JavaFX-based music diary and rating application that allows users to keep track of the songs they listen to, manage a personalized "Listen List", and rate tracks. 
The project was designed with an emphasis on user friendly features, and a structured MVC design.

Features
User & Navigation
- A login screen with basic credential handling
- Main menu options that direct users to different screens and features
- User profile page that displays user info and listen lists

Music Management
- Song list display
- Song search functionality
- Ability to add or remove tracks from user listen lists
- Info pertaining to users ratings and general info about songs and artists

Rating System
- Rate songs on a 1-5 star scale
- Update or remove ratings
- Personal rating history daved for each user

Data Handling
- CSV files used for storing users, songs, and ratings
- Clean separation of Model, View, and Controller classes

Interface
- Built fully using JavaFX and FXML
- Multiple screens (Login, Menu, User Screen, Rating Screen, Listen List)
- Simple UI with Roadrunner themed styling

Rowdy Rhythms was created to give users a simple and organized space to track their music habits.
